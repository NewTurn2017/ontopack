local-first · portable · agent-ready

Build knowledge packs you can actually move.

OntoPack turns Markdown, assets, citations, and media enrichment into a local folder that can be searched, served, bundled, restored, and exposed to AI agents through MCP.

Start in 5 minutes GitHub MVP runbook 
$ pack init research-pack
$ cd research-pack
$ cp ~/Downloads/slide.png _inbox/
$ pack process && pack build --incremental
$ pack search "ontology" --mode keyword

[keyword] Knowledge Graph Primer
  note: graph-primer / graph-primer#0000
  snippet: ontology, relation, entity...

Readable by default

Markdown and assets remain the source of truth. The database is a rebuildable derived index.

MCP without mystery

The MCP server returns context blocks and citations. Your agent writes the final answer.

Portable across machines

Bundle a pack into a directory or tar.gz archive, import it elsewhere, then rebuild locally.

Install from GitHub

macOS / Linux

git clone https://github.com/NewTurn2017/ontopack.git
cd ontopack
cargo build --release -p pack-cli
./target/release/pack --help
scripts/install.sh --prefix "$HOME/.local" --completion-shell zsh

Windows / Parallels

git clone https://github.com/NewTurn2017/ontopack.git
cd ontopack
cargo build --release -p pack-cli
.\target\release\pack.exe --help
powershell -ExecutionPolicy Bypass -File .\scripts\windows-smoke.ps1 -PackBin .\target\release\pack.exe

English 5-minute tutorial

1

Create

pack init ~/ontopack-demo
cd ~/ontopack-demo
2

Capture

cat > _inbox/source-card.md <<'NOTE'
---
type: note
title: Source Cards
tags: [research, citation]
---
A source card keeps useful context and citations.
NOTE
3

Search and serve

pack process
pack build --incremental
pack search "citation"
pack open --port 0 --no-browser --print-url
4

Move it

pack bundle ../demo-bundle --archive ../demo.tar.gz
pack init ../restored-pack
cd ../restored-pack
pack import ../demo.tar.gz

MCP and media workflows

Agent context

cargo build --release -p pack-mcp
pack-mcp --pack-root /path/to/pack

Use `search`, `ask`, `related`, `timeline`, and media tools from Claude/Codex.

Media enrichment

pack enrich-pending \
  --provider-command scripts/providers/fixture_media_worker.py \
  --limit 10

Provider workers read JSON from stdin and write safe enrichment patches to stdout.
